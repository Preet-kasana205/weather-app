🌦️ Weather App (Python)

A simple command-line weather application built using Python that fetches real-time weather data and a 5-day forecast using an external API.

🚀 Features:-
🌍 Get current weather of any city
📅 5-day weather forecast
📋 Menu-based system for easy navigation
🔐 Secure API key handling using .env

---------------------------------------------------------------
🛠️ Tech Stack
Python
Requests library
Dotenv (for environment variables)
Weather API (OpenWeather )

---------------------------------------------------------------

📦 Installation & Setup:-

Clone the repository:

git clone https://github.com/your-username/weather-app.git
cd weather-app

* Install dependencies:

pip install -r requirements.txt

* Create a .env file:

API_KEY=your_api_key_here

*Run the app:

python weather.py

⚠️ Important
Do NOT share your .env file
Keep your API key private
